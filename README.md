##使用
引入music.js,jquery与music.css<br>
调用createMusicPlayer<br>
```js
createMusicPlayer({
                  musicList :'音乐列表数组',
                  musicName: '音乐名对应字段名',
                  musicPath :'音乐路径对应字段名',
                  apiHost :'音乐数据获取域名',
                  musicBackground: '音乐背景图对应字段名(可选参数)' 
              })
```
